My Story

Hi, I’m Dave Irwin. A few years ago, I owned a successful business that built niche websites using WordPress. In fact, I created a network of websites that generated almost 100,000 monthly page views through content marketing and sound SEO practices. I was extremely pleased with that success, but I also learned something about myself: I really loved web development.

This wasn’t an entirely new revelation. I’d been fiddling with code since I was a teenager when I started programming in BASIC and using the command line. However, I now had a nascent desire to make it a career. Eventually I sold my business and went back to school to learn full stack development.

First, I became an expert with front-end technologies: HTML, CSS, JavaScript, Responsive Design, and React. Now, I’m completing a software engineering bootcamp focused on back-end technologies like Java, mySQL, and Web APIs with Spring Boot.

I’m on the lookout for my next great company. I want to work with a team that values broad real-life experience, and who’s willing to take a chance on a newly minted, but extremely capable, hard-working, & hungry software developer.

Are you that team? Please reach out and let’s talk.
